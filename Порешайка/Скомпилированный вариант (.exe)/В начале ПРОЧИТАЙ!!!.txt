Программа ПОРЕШАЙКА 

Программа написана для детей 1-3 классов для решения примеров по математике.

ВНИМАНИЕ!
   Возможно, ваша система как-то отреагирует на попытку запустить файл 
Порешайка_х64.exe или Порешайка_х86.exe. Этого бояться не надо!
Просто программа упаковывается в файл .exe для того, чтобы её можно было 
запускать на разных компьютерах без установки Python.
   Файл .exe с программой ПОРЕШАЙКА никуда и ничего в систему не устанавливает 
и не может её вывести из строя!!! При запросе - разрешите распаковку и запуск.

ДЛЯ РАБОТЫ С ПРОГРАММОЙ:
   1. Распакуйте архив Порешайка.zip на любой локальный диск (D, E и др.). 
Запустите файл Порешайка_х64 или Порешайка_х86 в зависимости от разрядности
вашей системы. Если у вас Windows 64bit, то запустите Порешайка_х64, если 
Windows 32bit - то Порешайка_х86.
   2. После запуска программы откроется основное окно, в котором надо будет
ввести число-ограничение и поставить флажки в нужных полях.
   3. Число-ограничение - это число, больше которого программа не будет 
генерировать числа в примерах. Т.е. если, например, число-ограничение = 20, 
то числа в примерах будут не более 20.
   4. Флажки в полях ставите по своему усмотрению. Если поставите только один 
какой-то флажок, то программа будет генерировать примеры только из выбранной 
области математики. Если поставите несколько флажков, то в случайном порядке
будут генерироваться примеры из всех выбранных вариантов.
   Флажки в выбранных вариантах можно менять по своему усмотрению в любое время, 
но должен стоять хотя бы один какой-то флажок.
   5. После ввода числа-ограничения и установки флажков в нужных полях - нажмите
кнопку "Нажми и появится пример", введите ваш ответ и нажмите кнопку "Проверить".
   6. При неправильном ответе можно его исправить и проверить новый ответ. 
Для этого удалите неправильный ответ, сразу введите новый вариант ответа и 
нажмите кнопку "Проверить". Если не хотите исправлять неправильный ответ, 
то нажмите кнопку "Нажми и появится пример" для генерации нового задания.
   7. Программа генерирует задания неограниченное количество раз. Для завершения
работы просто закройте программу.

P.S. При первом запуске программа может немного задуматься ))) Потом все будет ОК!
